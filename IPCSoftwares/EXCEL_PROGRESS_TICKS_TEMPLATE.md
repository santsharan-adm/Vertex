## ?? EXCEL FORMULAS (For Automated Ticks)

### Formula 1: Conditional Check Mark
```excel
=IF(D2="DONE","?","")
```

### Formula 2: Progress Bar (as percentage)
```excel
=COUNTIF(D:D,"DONE")/COUNTA(D:D)
```

### Formula 3: Status Badge
```excel
=IF(E2=1,"? COMPLETE","? PENDING")
```

### Formula 4: Phase Summary
```excel
=PHASE 1: &COUNTIF(A:A,"1")&" of 6 tasks ?"
```

## ?? HOW TO USE IN EXCEL

### **Step 1: Create Headers**
```
Column A: Phase
Column B: Task #
Column C: Task Description
Column D: Status
Column E: Completion %
Column F: Date
Column G: ? Ticks
```

### **Step 2: Add Data**
Copy data from Option 2 above into Excel

### **Step 3: Add Conditional Formatting**
- Select Column D
- Format > Conditional Formatting
- Rule: If cell value = "DONE" ? Fill GREEN
- If cell value = "VERIFIED" ? Fill BLUE
- If cell value = "CONFIRMED" ? Fill BLUE

### **Step 4: Add Checkmarks**
- Column G formula: `=IF(D2="DONE","?","")`
- Copy down for all rows

- Add summary metrics from Option 4
- Use COUNTIF formulas to auto-calculate
