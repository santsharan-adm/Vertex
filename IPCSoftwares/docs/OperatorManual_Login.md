# AOI System Operator Manual (Login Module)

> **Project:** IPCSoftware.app
> **Scope:** Operator-facing steps to sign in and access the application shell.
> **Audience:** Line Operators / Supervisors

---

## 1. Prerequisites
- Valid username and password provisioned by Admin.
- Application installed and reachable; PLC connectivity not required for login.
- If using Windows kiosk mode, ensure keyboard is available for input.

## 2. Launch Application
1. Start the AOI System application (`IPCSoftware.app`).
2. Wait for the login screen to appear.

## 3. Enter Credentials
1. Click the `Username` field and type your username.
2. Click the `Password` field and type your password.
3. (Optional) Confirm focus is on the username field by default.

> _Screenshot placeholder_: **Entering Credentials**
> (Insert image with sample username/password entry.)

## 4. Sign In
1. Click the `Login` button.
2. On success, the app loads the Dashboard and the top Ribbon.
3. Your display name appears on the right side of the Ribbon.

> _Screenshot placeholder_: **Post-Login Dashboard**
> (Insert image showing the dashboard and ribbon after successful login.)

## 5. Handling Login Errors
- **Invalid username/password:** Re-enter credentials. If it persists, contact your Admin to reset the password.
- **Account inactive/locked:** Contact Admin to reactivate.
- **Network issues / app cannot reach services:** Retry; if still failing, escalate to IT/Maintenance.

## 6. Logout
1. Click the `Logout` button on the Ribbon (right side).
2. Confirm if prompted.

> _Screenshot placeholder_: **Logout Button**
> (Insert image highlighting the logout control on the Ribbon.)

## 7. Roles and Access
- **Operator:** Standard access; limited configuration.
- **Admin:** Can configure users, logs, devices, alarms, reports.

## 8. Quick Tips
- Ensure Caps Lock is off when typing passwords.
- If the app window is minimized, restore it from the taskbar.
- Keep credentials confidential; never share with others.
